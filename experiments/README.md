# Experiments

This directory hosts experiment orchestration (QASecClaw + baselines) and evaluation:

- `pipelines/`: runnable experiment scripts (WIP)
- `baselines/`: wrappers and normalizers for Semgrep / ZAP / Snyk, etc. (WIP)
- `metrics/`: metric calculators and aggregation conventions (WIP)
- `python/`: Python utilities for sampling/metrics/plots/log analysis

Outputs should be written under `artifacts/` (gitignored).
