# Datasets

This repo does **not** vendor large datasets.

Use `datasets/configs/` to define where datasets are located locally and what subset to sample.

## Supported datasets (planned)

- Security: OWASP Benchmark, DVWA, WebGoat
- QA bugs: Defects4J, Bugs.jar, CodeXGLUE
- Logs: LogHub (HDFS, BGL, Thunderbird)
- Web interaction: WebArena
- Optional network security: CICIDS2017, UNSW-NB15
